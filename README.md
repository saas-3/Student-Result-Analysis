# Student Result Analysis

This project focuses on analyzing student performance data using Python. The goal was to understand how factors such as gender, parental education level, marital status, study hours, and ethnic background influence academic scores in Math, Reading, and Writing.

## What I Implemented
- Loaded and cleaned the dataset using Pandas  
- Checked dataset structure and missing values  
- Analyzed gender distribution  
- Performed groupby aggregation to study parental education impact  
- Created heatmaps to visualize relationships between variables  
- Used boxplots to examine score distribution and outliers  
- Analyzed ethnic group distribution using a pie chart  

## Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn

## How to Run
Install required libraries:
pip install numpy pandas matplotlib seaborn

Run the script:
python analysis.py

This project demonstrates my understanding of basic Exploratory Data Analysis (EDA), data visualization, and handling real-world datasets using Python.
